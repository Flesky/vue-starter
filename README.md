# Vue 3 starter

A simple starter project for Vue 3.

## Features:

- Vue 3
- File based routing
- Auto-imported APIs, components, and icons
- Tailwind CSS
- TypeScript
- ESLint

## How to use:

- [ ] Clone this repo
- [ ] Add name and version to package.json
- [ ] [Configure your icons](https://github.com/antfu/unplugin-icons)
